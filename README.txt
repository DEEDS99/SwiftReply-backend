# ⚡ SwiftReply – WhatsApp Automation SaaS

Production-ready WhatsApp automation platform built with FastAPI + React, deployable to Render.com.

## 🚀 Quick Deploy to Render

1. **Fork/push this repo to GitHub**
2. **Connect to Render.com**: Go to [render.com](https://render.com) → New → Blueprint
3. **Select your repo** — Render reads `render.yaml` and auto-creates all services
4. **Set environment variables** (see below)
5. **Deploy** — all services spin up automatically

## ⚙️ Environment Variables

### Backend (Required)
| Variable | Description |
|---|---|
| `DATABASE_URL` | PostgreSQL connection string (auto-set by Render) |
| `REDIS_URL` | Redis connection string (auto-set by Render) |
| `SECRET_KEY` | JWT secret key (auto-generated by Render) |
| `FRONTEND_URL` | Frontend URL for CORS (auto-set by Render) |
| `WHATSAPP_VERIFY_TOKEN` | Token for WhatsApp webhook verification |

### Backend (Optional)
| Variable | Description |
|---|---|
| `WHATSAPP_API_TOKEN` | Meta WhatsApp Business API token |
| `WHATSAPP_PHONE_NUMBER_ID` | WhatsApp Phone Number ID from Meta |
| `OPENAI_API_KEY` | OpenAI API key for AI-powered replies |
| `STRIPE_SECRET_KEY` | Stripe secret key for payments |
| `STRIPE_WEBHOOK_SECRET` | Stripe webhook signing secret |
| `FLUTTERWAVE_SECRET_KEY` | Flutterwave secret key |
| `SENTRY_DSN` | Sentry error tracking DSN |

## 🔧 Local Development

### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env       # Edit with your values
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
cp .env.example .env.local  # Set VITE_API_URL=http://localhost:8000/api/v1
npm run dev
```

### Celery Worker (optional for background tasks)
```bash
cd backend
celery -A app.tasks.celery_app worker --loglevel=info
```

## 📱 WhatsApp Setup

1. Create a Meta Developer account at [developers.facebook.com](https://developers.facebook.com)
2. Create a WhatsApp Business app
3. Get your **Phone Number ID** and **API Token**
4. In your Meta App settings, configure the webhook:
   - **URL**: `https://your-api.onrender.com/api/v1/webhook/whatsapp`
   - **Verify Token**: `swiftreply_verify_token` (or your custom value)
   - **Subscribe to**: `messages`

## 💳 Payment Setup

### Stripe
1. Get keys from [dashboard.stripe.com](https://dashboard.stripe.com)
2. Create products/prices for each plan
3. Update `stripe_price_id` in `payment_service.py`
4. Configure webhook endpoint in Stripe dashboard

### Flutterwave
1. Get keys from [dashboard.flutterwave.com](https://dashboard.flutterwave.com)
2. Add `FLUTTERWAVE_SECRET_KEY` and `FLUTTERWAVE_PUBLIC_KEY` to env
